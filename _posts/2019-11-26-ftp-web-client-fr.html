Ils utilisent Filestash






Accès à votre serveur FTP en ligne

Notre client FTP en ligne est facile à utiliser. Il est compatible avec des serveurs comme proftpd, pureftpd, filezilla, vsftpd et tous les autres serveur FTP offrant une interface FTP.

Éditeur de contenu

L'éditeur FTP intégré vous permet de modifier directement vos fichiers en ligne. Il reconnait un large éventail de type de fichier.

Logiciel libre et open source

Filestash est 100% open source. Le code est disponible sur github. Allez faire un tour et héberger votre propre version (voir la documentation).

Mac, Windows and Linux

Notre outil est disponible sous la forme d'une application web en ligne ce qui vous donne la portabilité ultime sans avoir à installer quoi que ce soit.

Partage de fichiers et recherche

Vous pouvez facilement faire des liens partagés en toute sécurité. De plus, le moteur de recherche vous permet de trouver facilement vos données.

Mettre votre FTP sur le web

Vous pouvez partagé vos données directement sur votre site via une iframe avec un lien de partage comme nous l'avons fait dans notre page d'accueil

En ligne et hors ligne

Filestash est un PWA, ce qui signifie qu'il fonctionne non seulement très bien en ligne mais qu'il possède également certaines fonctionnalités pour une utilisation hors ligne.

S'intègre à votre gestionnaire de fichiers

Filestash s'intègre à votre gestionnaire de fichiers natif via le protocole Webdav

S'étend en fonction de vos besoins

Notre outil comporte un mécanisme de plugin permettant de faire des extensions. Si cela vous interesse, allez voir la documentation

Notre objectif: developper le meilleur gestionnaire de fichier universel

Les clients FTP traditionnels tels que Filezilla FTP or WinSCP sont très bien pour les administrateurs systèmes et cela les rends difficile à utiliser pour le reste du monde. Filestash s'attache à rendre FTP aussi facile à utiliser que Dropbox tous en respectant votre vie privée.
