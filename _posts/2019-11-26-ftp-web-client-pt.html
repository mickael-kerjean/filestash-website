Podemos integrar o Filestash para trabalhar com seu backend, seus processos, seus usuários e sua marca Fale com a gente

Explore e gerencie seu servidor FTP

Nosso cliente web FTP facilita a gestão e exploração dos dados em seu servidor. Ele funcionará com qualquer servidor compatível com FTP, como Proftpd, Pureftpd, Filezilla, Vsftpd, ...

Editor de conteúdo e visualizador

Assista seus vídeos ou imagens, o editor FTP integrado permite que você visualize e edite não apenas documentos do Office, mas também uma ampla gama de formatos de arquivo e muito mais

Código aberto e Software Gratuito

Filestash é um software de código aberto. Nosso código está disponível no Github para que você possa auditar, expandir e auto-hospedar sua própria versão (veja a documentação).

Mac, Windows e Linux

Nossa ferramenta de software está disponível como um aplicativo web do seu navegador dando a você a portabilidade final sem ter que instalar nada. Funciona melhor com Chrome, Chromium e Firefox.

Compartilhamento de Arquivos e pesquisa completa de texto

Colabore com um link compartilhado que permanece sob controle rigoroso. O mecanismo de pesquisa completa de texto torna fácil encontrar o conteúdo existente.

Exponha seu FTP na internet

Ofereça aos seus usuários a interface que eles mereciam e integre através de um iframe ou com uma página personalizada como esta página

Online e offline

Como um PWA, Filestash não só funciona muito bem online, mas também vai mais longe para fornecer algumas funcionalidades básicas para o uso offline

Integra com seu gerenciador de arquivos

Links compartilhados se integram ao seu gerenciador de arquivos. Eles podem ser usados como uma unidade de rede para windows, OSX ou Linux. Esses links são um servidor WebDAV completo que se liga ao seu FTP

Extensível

Nossa ferramenta web FTP pode ser totalmente personalizada. Pode ser rotulada em branco e estendida através do mecanismo de plugin para atender às suas necessidades e processos com muita flexibilidade

Criando um incrível cliente Web FTP

Clientes FTP tradicionais, como Filezilla FTP ou WinSCP, são ótimos para administradores de sistemas e engenheiros, mas todos os seus usuários podem não estar familiarizados com o protocolo subjacente e apenas querem uma ferramenta para fazer as coisas de uma maneira simples. A proposta de valor do Filestash é tornar o FTP fácil de usar para seus usuários finais, trazendo recursos de colaboração para tornar FTP uma substituição moderna para Dropbox.
